# 카톡 파일 파싱 Skill

## 역할
카톡 내보내기 txt 파일을 읽고 정제합니다.

## 정제 항목

### 제거할 것
- "[닉네임]님이 들어왔습니다"
- "[닉네임]님이 나갔습니다"
- "[사진]", "[동영상]", "[이모티콘]"
- "메시지가 삭제되었습니다"
- "관리자가 메시지를 가렸습니다"
- 오픈채팅봇 메시지 (공지용)
- 카카오톡 시스템 메시지

### 남길 것
- [닉네임] [시간] 실제 대화 내용
- 모든 사용자의 일반 메시지
- 링크 (URL)
- 의미있는 모든 대화

## 출력 형식

```json
{
  "fileName": "싱크클럽_초보방.txt",
  "saveDate": "2026-03-09",
  "dateRange": {
    "start": "2026년 2월 16일",
    "end": "2026년 3월 9일"
  },
  "totalMessages": 150,
  "cleanedMessages": 120,
  "messages": [
    {
      "date": "2026년 2월 16일 월요일",
      "nickname": "클로드 초보요",
      "time": "오후 11:27",
      "content": "아 ㅠㅠ"
    }
  ]
}
```

## 사용 예시

```
이 카톡 파일 정제해줘. 불필요한 메시지 전부 제거하고.
```
