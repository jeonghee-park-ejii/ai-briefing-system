# 결과 포맷팅 Skill

## 역할
분석된 데이터를 정희쌤 브랜드에 맞는 마크다운 보고서로 정리합니다.

## 출력 포맷 (마크다운)

```markdown
# 📌 AI 단톡방 브리핑 - YYYY.MM.DD

## 🔥 핫토픽 1 | 제목

상세한 내용을 3-5줄로 정리합니다.
어떤 대화가 오갔는지, 왜 중요한지 설명합니다.

📎 관련 기사
- [제목](URL) - 설명
- [제목](URL) - 설명

---

## 🔥 핫토픽 2 | 제목

(같은 형식)

---

## ❓ Q&A 모음

Q. 바이브코딩 시작하는 방법?
A. Claude Code 설치 후... (상세 답변)

Q. 다른 질문?
A. 답변...

---

## 🔗 공유된 주요 링크

⭐ 중요
- [제목](URL) - 설명

일반
- [제목](URL) - 설명

---

**분석일**: YYYY-MM-DD
**새로운 항목**: N개
**업데이트된 항목**: N개
**제거된 중복**: N개
```

## 엑셀용 요약 (3-5줄)

마크다운 보고서와 별도로
엑셀에 들어갈 요약본도 생성합니다.

예:
Claude 4 새 기능 출시
OpenClaw 보안 방법 논의
바이브코딩 입문자 팁

## 사용 예시
이 분석 결과를 마크다운 보고서로 정리해줘.
정희쌤 브랜드처럼 깔끔하게.
엑셀용 요약도 3-5줄로 만들어줘.
